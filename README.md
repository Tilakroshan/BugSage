# 🐞 BugSage — AI Bug Detection Assistant

BugSage is a full-stack AI-powered tool that helps developers detect bugs in code and receive intelligent suggestions for fixing them.


## 🚀 Why BugSage?

Debugging is time-consuming. BugSage simplifies this by:
- Automatically identifying errors
- Providing AI-generated fixes
- Improving developer productivity


## ✨ Features

- 🔍 Detects bugs in code  
- 💡 Suggests fixes using AI  
- ⚡ Fast and simple interface  
- 🌐 Full-stack application (Frontend + Backend)  


## 🛠️ Tech Stack

- **Frontend:** React (Vite), JavaScript  
- **Backend:** Node.js, Express  
- **AI Integration:** OpenAI / Gemini API

##⚙️ Setup

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/BugSage.git
cd BugSage
```

## 2.SETUP FRONTEND
cd frontend
npm install
npm run dev

## 3.SETUP BACKEND
cd backend
npm install
npm run dev

## Environment Variables
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-2.5-flash

##🧑‍💻 Usage
Open the app in your browser at http://localhost:5173
Paste your code into the editor
Click Detect Bugs
View AI-generated bug reports and fix suggestions
